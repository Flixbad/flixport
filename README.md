# FlixDev - Portfolio

Portfolio personnel présentant mes projets de développement web, mon parcours professionnel et mes passions.

## Technologies utilisées

- Angular 17
- TypeScript
- SCSS

## Installation

```bash
npm install
```

## Développement

```bash
npm start
```

L'application sera accessible sur `http://localhost:4200`

## Build de production

```bash
npm run build:prod
```

Les fichiers seront générés dans le dossier `dist/flixdev`

## Déploiement

Le projet peut être déployé sur un VPS en copiant le contenu du dossier `dist/flixdev` vers le serveur web.
